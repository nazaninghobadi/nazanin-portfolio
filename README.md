# 🌐 Nazanin Ghobadi – Portfolio

This repository contains my personal portfolio website, showcasing my projects, achievements, and learning journey in Artificial Intelligence and Machine Learning.

The website was initially generated with the assistance of AI and then customized by me to reflect my own projects, experience, and personal style. While my primary programming language is Python, this project also gave me practical exposure to working with a modern React-based frontend.

## 🚀 Live Demo

https://nazanin-portfolio.nghobadi.workers.dev/

## ✨ Features

- Responsive and modern interface
- Project showcase with external links
- Technical skills overview
- Achievements and activities
- Contact section

## 🛠️ Technologies Used

- React
- Vite
- Tailwind CSS v4
- Framer Motion

## 📦 Run Locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## 🏗️ Build

```bash
npm run build
```

The production-ready files will be generated in the `dist/` directory.
